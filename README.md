## üõ† Installation Guide

### 1Ô∏è‚É£ System Management (Laravel)

1. Download and extract the files into your public hosting directory
2. Import the database file: `inventory_store.sql`
3. Configure your database settings in the `.env` file
4. Visit:
   `http://YourDomainName/inventory-stock-system/public/login`
5. System setup complete

---

### 2Ô∏è‚É£ Android Application

1. Build the Android project using Android Studio
2. Install the app on your Android device
3. Open **Settings** in the app
4. Enter:

   * Backend system URL
   * API KEY (available in **Company Details** after logging into the system)
   * Or scan the QR code for automatic setup

5. Installation complete
